Taller de Colisiones 2D

Simulación en Python + Pygame de detección de colisiones entre círculos y rectángulos.

Ver en GitHub Cómo ejecutarlo

Demo

Instrucciones

Clona el repositorio y ejecuta la simulación en tu PC:

git clone https://github.com/lBIGhellboyl07/TALLER-COLISIONES/blob/main/TALLERCOLISIONES.py
cd TALLER-COLISIONES
pip install -r requirements.txt
python main.py

Controles

  • ESPACIO → Iniciar simulación
  • C → Cambiar método (Fuerza Bruta / AABB)
  • P → Volver al menú
  • ESC → Salir
